Conhecimentos

HTML

HTML é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo de uma página.

Javascript

JavaScript é uma linguagem de programação criada para ser executada em navegadores e manipular comportamentos de páginas web.

CSS

Trata-se de uma linguagem de marcação, amplamente utilizada com HTML ou XHTML, e representa diversas possibilidades para a formatação.

Bootstrap

Bootstrap é um framework web aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

Design

O design é um conceito amplo que representa o processo de criação de produtos a partir da excelência técnica e estética, com o objetivo de solucionar problemas e agregar valor.

Git e GitHub

O GitHub é o maior repositório de dados compartilhado do mundo, com 28 milhões de usuários atualmente. É uma plataforma majoritariamente usada por desenvolvedores.
